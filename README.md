# krandom
Java Random Utils for Testing.

# Usage
## Import:
<p><code>import org.kedos.KRandom;</code></p>
or
<p><code>import static org.kedos.KRandom.*;</code></p>

## Methods:
<p><code>
String babyTalk = KRandom.nextAlphabeticString(n);
</code></p>
<p><code>
int[][] matrix = KRandom.next2DIntArray(n, m);
</code></p>
